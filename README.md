# Gulp
### Basik structure for projects

## It can:
- Build html file from other html files
- Convert scss to css
- Convert js
- Reduce images
- Convert fonts: ttf -> woff2

## Command for start
- git clone https://github.com/Alex-Hlatsko/Gulp.git
- cd Gulp
- npm i (npm install)
- npm run dev (for development)
### or 
- npm run build (for production)
